CHC Web

Repository documentation portal

This GitHub Pages target publishes repository documentation only. The live Next.js application is prepared for Docker and Helm deployment.

Current stack

Next.js 16, React 19, TypeScript, Tailwind CSS v4, shadcn/ui, Prisma v7 preparation, `next-intl`, Docker, Helm, and `pnpm`.

Key rules

  • pnpm only
  • Atomic design for project components
  • shadcn/ui for reusable UI primitives
  • Static mock content in src/lib/content until backend integration
  • YAML files use the .yaml extension only
  • Repository docs live in docs/
